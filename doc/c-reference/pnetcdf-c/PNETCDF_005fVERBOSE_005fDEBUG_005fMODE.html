PNETCDF_VERBOSE_DEBUG_MODE

Available in 1.7.0 and later.

PNETCDF_VERBOSE_DEBUG_MODE environment variable can be used to print the location in the source code where the error code is originated, no matter the error is intended or not. This run-time environment variable only takes effect when PnetCDF is configure with debug mode, i.e. --enable-debug is used at the configure command line. Users are warned that enabling this mode may result in a lot of debugging messages printed in stderr.

Set this variable to 1 to enable. Set it to 0 or keep it unset disables this mode. Default is 0, i.e. disabled.
